This project focuses on analyzing hospital patient data to generate actionable insights for healthcare decision-making. SQL is used for data extraction and analysis, while Power BI is used for interactive dashboards and visualizations.

Key Highlights

Patient demographics analysis (age, gender, location)

Disease-wise patient distribution

Admission and discharge trend analysis

Treatment cost and billing insights

Hospital performance evaluation

Tools & Technologies

SQL (MySQL)

Power BI

Python (EDA using Pandas)

CSV Healthcare Dataset

Project Outcome
The analysis helps understand patient trends, optimize hospital operations, and support data-driven decisions through clear visual insights.
